# Kairos Platform

This folder stores the long-term Kairos platform architecture and implementation records.

## Subfolders

```text
command-centers/       Major Kairos command centers and vertical slices.
admin-workspace/       Admin Operations Center and internal operator workflows.
customer-workspace/    Customer-facing portal workflows and customer lifecycle logic.
data-models/           Entities, schemas, relationships, and shared operating records.
automation/            Automation and intelligence workflows.
growth-marketing/      Growth, marketing, advertising, campaign, and promotion systems.
quality-release/       Quality, release, validation, and deployment operating systems.
production/            Production Command Center workflows and manufacturing logic.
```

## Direction

Kairos is the flagship operating platform for MMG. This folder should evolve from architecture into production-ready implementation packages.
