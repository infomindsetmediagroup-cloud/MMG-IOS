# Releases

This folder stores release packages, QA records, deployment summaries, changelogs, and release notes.

## Subfolders

```text
packages/      Release packages grouped by feature, page, product, or platform slice.
qa/            QA reports, link checks, validation notes, and regression checks.
changelog/     Chronological changelog records.
deployments/   Deployment summaries and live-environment notes.
```

## Rule

Every meaningful production change should have a traceable release note or QA reference.
