# MMG Web App Surface Map

## Public Surface
Purpose: introduce the MMG ecosystem and route users into books, creator education, AI tools, subscriptions, and Kairos.

Initial modules:
- Hero
- Featured products
- Knowledge Library preview
- Kairos introduction
- Creator education pathways
- Founder story
- Final call to action

## Customer Surface
Purpose: give authenticated customers a personalized operating space.

Initial modules:
- Customer dashboard
- Purchased downloads
- Subscription status
- Recommended knowledge
- Kairos assistant panel
- Project workspace
- Content review window

## Admin Surface
Purpose: give MMG executive operations a controlled internal command center.

Initial modules:
- Executive dashboard
- Product publishing queue
- Shopify operations
- Customer intelligence status
- Kairos runtime health
- Trust Layer audit stream
- Release checklist

## API Surface
Purpose: expose controlled server-side runtime endpoints.

Initial endpoints:
- Kairos runtime route
- Health route

Future endpoints:
- Customer vault
- Publishing products
- Admin releases
- Trust audit
