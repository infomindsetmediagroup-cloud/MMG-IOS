# Shopify Product Value Block Install Notes

## Files

- `shopify/products/customer-value-product-block.html`
- `shopify/products/customer-value-product-block.css`

## Purpose

This package adds a reusable product-page block that connects each MMG product to the larger MMG / Kairos customer-value system.

The block prevents product pages from reading like isolated purchases. It frames books, digital downloads, templates, guides, and service products as practical tools inside the broader path of discovering, organizing, building, preserving, and sharing customer knowledge.

## Recommended Placement

Place this block below the main product description and above related products, reviews, or footer sections.

For service products, place it near the onboarding explanation so the customer understands how the purchase connects to guided execution.

For digital downloads, place it after the product details so the customer understands how to apply the download as part of a larger body of work.

## Message Hierarchy

1. Product as part of the MMG / Kairos system.
2. Product as a practical action tool.
3. Learn / Apply / Preserve / Build Forward pathway.
4. Guardrail that avoids guaranteed outcome or shortcut claims.

## Guardrails

Do not position products as:

- guaranteed income tools,
- overnight-success shortcuts,
- passive-income promises,
- isolated downloads with no next step,
- publishing-company-first offers,
- AI-tool-first offers.

Approved positioning:

- practical education,
- guided execution,
- durable assets,
- customer capability,
- long-term value,
- useful next actions.

## Shopify Use

Use the HTML in a custom-liquid or custom-HTML product-page block. Use the CSS in theme custom CSS, a matching asset file, or the same block if the theme allows scoped styling.
