# Shopify Customer Value Hero Install Notes

## Files

- `shopify/homepage/customer-value-hero.html`
- `shopify/homepage/customer-value-hero.css`

## Purpose

This package gives the MMG Shopify homepage a customer-facing value-first hero that leads with the canonical promise:

**Your Knowledge Has Value.**

Supporting statement:

**Helping you discover it, build it, and share it with the world.**

## Installation Path

Use the HTML as the homepage custom-liquid or custom-HTML block source. Use the CSS as the matching style layer in the same block, theme custom CSS area, or theme asset file depending on the active Shopify theme workflow.

## Required Message Hierarchy

1. Customer value promise.
2. Supportive guidance statement.
3. Practical explanation of MMG and Kairos.
4. Opportunity / Asset / Audience / Execution pathways.
5. CTAs to Kairos and resources.
6. Guardrail language that avoids shortcut or guarantee framing.

## Guardrails

Do not rewrite the hero into:

- get-rich-quick language,
- guaranteed income language,
- passive-income claims,
- hype-first positioning,
- publishing-company-first positioning,
- AI-tool-first positioning.

The customer-facing voice should be practical, supportive, strategic, and value-building.

## CTA Defaults

Primary CTA:

`Explore Kairos` → `/pages/kairos`

Secondary CTA:

`Browse Resources` → `/collections/all`

These may be adjusted only if the live Shopify route structure changes.
