# Assets

This folder stores reusable MMG and Kairos asset records.

## Subfolders

```text
brand/        Logos, identity notes, and approved visual standards.
images/       Image source references and production-ready image notes.
prompts/      Image, video, product, ad, and content-generation prompts.
covers/       Book covers, blog covers, product covers, and promotional covers.
social/       Short-form and social content asset records.
```

## Production Note

GitHub preserves source assets and references. Live customer-facing image delivery may still use Shopify CDN or another production CDN when performance matters.
