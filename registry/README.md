# Registry

This folder stores canonical MMG and Kairos operating registries.

## Subfolders

```text
pages/          Public site page registry and page status records.
products/       Product registry, product handles, source status, and launch records.
campaigns/      Marketing, advertising, promotion, and campaign registry.
releases/       Release index and release status records.
assets/         Asset registry and source-location records.
editorial/      Editorial publishing calendar, topic registry, and content status.
```

## Purpose

Registries prevent duplication, preserve current status, and make future production work traceable.
