# Parent Card Completion Gate

Production objective: finish the five command-center parent cards before any additional capability work.

## Canonical parent centers

1. Knowledge
2. Content
3. Business
4. Customers
5. Operations

## Completion contract

A parent center may report COMPLETE only when all five canonical child entry points:

- render in the center workspace;
- open a real governed destination or control surface;
- provide actionable empty, loading, success, and failure states where applicable;
- preserve return navigation to the command center;
- expose verified readiness evidence rather than a manually inflated percentage;
- pass browser and mobile command-center validation.

## Scope lock

Until all five parent centers satisfy this contract:

- no new Shopify analytics enhancements;
- no unrelated dashboard modules;
- no new capability surfaces outside the five-center hierarchy;
- production work is limited to parent-card completion, regression repair, and deployment reconciliation.
