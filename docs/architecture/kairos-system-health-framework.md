# Kairos System Health Framework

Status: Approved architecture.

Purpose: Provide a single executive health scorecard that summarizes the operating condition of the entire MMG/Kairos ecosystem.

Health domains:
- Business
- Operations
- Production
- Publishing
- Customer Success
- Marketing & Growth
- AI Runtime
- Infrastructure
- Security
- Finance (when available)

For each domain report:
- Current status
- Trend
- Highest priority issue
- Recommended next action

The Overall Health Check aggregates these domains into one executive briefing before any drill-down views are shown.

Rule: A healthy system should require minimal executive intervention.
