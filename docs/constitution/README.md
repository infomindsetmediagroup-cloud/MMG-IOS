## Conflict rule

Where documents conflict, the highest applicable authority governs. Specialized documents govern their subject only when consistent with Constitution v1.0.

## Amendment rule

Constitutional changes require explicit executive approval, named scope, repository amendment, impact analysis, and formal refreeze. Ordinary implementation updates do not reopen the Constitution.
