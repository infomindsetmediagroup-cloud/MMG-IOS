# Canonical Command Wiring

Status: implementation contract
Version: 2026.07.17-v1

- The Command Center remains usable when optional workspace modules fail to load.
- Every non-website child card exposes a built-in objective executor that posts to `/api/hub/execute` with its authoritative action contract and instruction set.
- Dedicated domain workspaces are progressive enhancements, never the only execution path.
- Website Retool uses `docs/KAIROS_ENTERPRISE_WEBSITE_BUILD_INSTRUCTIONS.md` version `2026.07.15-v1`.
- Canonical Website Retool authorizes structural and visual changes only on the verified unpublished staging theme.
- Live MAIN publication remains prohibited until the rendered preview is explicitly approved and the separate release gate is completed.
- The browser shell has no top-level dependency on optional workspace modules.
