## Customer Portal Design Studio Capability
The Design Studio is the Customer Portal creative production layer. It provides the backend foundation for customer projects, assets, versions, exports, brand kits, AI-assisted generation/refinement, and Customer Knowledge Vault linkage.

Initial implementation should start after Command Center / command setter work is complete and should prioritize document/book projects, image/social asset projects, customer asset storage, version history, export jobs, and Kairos assistant hooks.
